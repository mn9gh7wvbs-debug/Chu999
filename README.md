锄九是一种源自广东地区的四人扑克游戏，玩法类似“锄大地”，使用1至9的36张牌，结合娱乐、休闲和竞技元素。
锄九起源于广东地区的传统民间扑克游戏，近年来在“三连一阳”（连州、连南、连山、阳山）等地流行。它不仅保留了传统玩法，还融入现代元素，使玩家在手机或线下聚会中都能体验地道的广东文化。 
牌张与大小
使用36张牌，牌面从1至9。
花色大小顺序：黑桃 > 红桃 > 梅花 > 方块。
单张牌大小示例：黑桃9 > 红桃9 > 梅花9 > 方块9 > 黑桃8，以此类推。
五张牌组合大小顺序：同花顺 > 四条 > 葫芦 > 同花 > 顺子。
游戏规则
人数：四人游戏，每人9张牌。
出牌顺序：持有最小牌（方块1）的玩家先出牌。
出牌形式：单张、对子、三张牌、五张牌。
特殊规则：如果手中有9且未出过1至8的牌，则9不能出。最后一轮若用9点牌收数，其他玩家需接受惩罚。
